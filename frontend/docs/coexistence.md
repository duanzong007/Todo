# 新旧前端并存策略

## 原则

所有主要页面已经切换到 Vue 生产路径；Go 继续负责登录、SSO 回调、JSON/API 和静态资源服务。

Vue 前端作为独立工程存在，负责首页、管理页、好友页和短信导入页的界面状态。

## 当前并存方式

- SSO 登录入口由 Go 后端服务：
  - `/login`
  - `/auth/sso/start`
  - `/auth/sso/callback`
- Vue 构建产物由 Go 服务在 `/` 提供
- Vue 构建产物由 Go 服务在 `/me` 和 `/me/friends` 提供
- Vue 构建产物由 Go 服务在 `/sms/native` 提供
- Vue + Vite 独立运行在：
  - `http://127.0.0.1:5173`
- Vite 通过代理访问 Go 后端：
  - `/dashboard`
  - `/events`
  - `/tasks`
  - `/imports`
  - `/sms`
  - `/me`
  - `/login`
  - `/auth`

## 迁移方式

### 阶段 1

只验证工程和接口，不接管任何页面。

### 阶段 2

优先迁移 `/me` 管理页。

当前方式：

- Vue 独立工程实现管理页
- Go 新增 `/me/data` JSON 数据端点
- Go 服务 `/me` 和 `/me/friends` 提供 Vue 构建产物
- 批量操作继续复用 `/me/tasks/apply`

### 阶段 3

迁移短信导入页。

注意：

- 保留安卓壳原生桥
- Web 端和壳端继续使用同一套页面
- Go 服务 `/sms/native` 提供 Vue 构建产物
- 本地历史缓存 key 保持不变

### 阶段 4

迁移首页任务区。

注意：

- 首页是高频核心页，已经切换到 Vue 构建产物
- Go 旧模板首页和 `/classic` 回退入口已经删除
- 任务列表、已完成列表、金句空状态、更多面板、添加面板、延期选择器和时间选择器都由 Vue 接管
- 任务业务规则仍然走 Go 后端接口

## 不变量

迁移期间必须保持：

- 生产 URL 可访问
- 现有 session cookie 继续可用
- PWA 与安卓壳不被破坏
- 后端业务语义不重复实现

## 当前状态

首页、管理页、好友页和短信导入页 Vue 已切换到生产路径；旧模板和迁移期兼容入口已经清理。
